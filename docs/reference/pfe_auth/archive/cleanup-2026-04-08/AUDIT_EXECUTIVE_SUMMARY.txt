WHAT THIS IS
============
AdaptIQ is an adaptive learning quiz platform using IRT (Item Response Theory)
to dynamically adjust question difficulty based on user performance.

CURRENT STATE: PARTIALLY COMPLETE
==================================
✅ Core systems working (100% test pass rate):
   - User authentication (JWT + bcrypt + rate limiting)
   - Classic room (question generation via RAG, adaptive difficulty)
   - Hint system (LLM-powered, answer-safe)
   - Session management (Redis-backed)
   - Database (PostgreSQL with migrations)
   - Server-side time tracking (prevents tampering)

❌ Major gaps (blocking MVP):
   - Challenge room: COMPLETELY MISSING (40 hours work)
   - Form validation: Frontend buttons unlocked during submission (2 hours)
   - Concept filtering: Questions not ranked by user's knowledge (8 hours)
   - Question repetition: No tracking of repeated questions (6 hours)
   - ELO decay: Inactive users don't lose rating (4 hours)
   - Test users: No varied concept knowledge profiles (2 hours)

CRITICAL PATH (3-4 WEEKS)
==========================
Week 1: Challenge Room + Form Locking (60 hours)
  - Database: 3 new tables (challenge_ranks, user_challenge_ranks, matches)
